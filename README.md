# liblognorm cookbook

Installs the C library [liblognorm](http://www.liblognorm.com//) for normalizing syslog events.

# Requirements

This cookbook depends on the `apt` cookbook.

# Recipes

default - installs apt packages for libee, libest, and liblognorm
